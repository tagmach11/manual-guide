## 기능 설명

#### 1. [목록](템플릿-수강신청서.md)  
수강신청서 목록 화면으로 돌아갑니다.

#### 2. 저장  
입력한 내용을 저장하여 수강신청서를 등록합니다.

#### 3. 삭제  
작성 중인 수강신청서를 삭제합니다.

#### 4. 순서변경  
항목의 표시 순서를 조정할 수 있습니다.

#### 5. 주소찾기  
주소 입력 시 우편번호 검색을 통해 쉽게 주소를 입력할 수 있습니다.

#### 6. 입력할 정보  
수강신청서에 필요한 기본 정보를 입력합니다.  
(예: 이름, 연락처, 이메일, 학번, 과목명 등)

#### 7. 서약서 입력  
수강신청 시 필요한 서약서 내용을 입력할 수 있습니다.  

#### 8. 수정가능 여부  
해당 신청서의 수정 가능 여부를 설정할 수 있습니다.

#### 9. 사용여부  
해당 신청서를 현재 사용(활성화)할지, 미사용(비활성화)할지 선택합니다.

---

## 사용 방법
- **목록 이동**: 기존 신청서 목록으로 되돌아가 관리합니다.  
- **저장**: 입력된 수강신청서 정보를 저장합니다.  
- **삭제**: 필요 없는 신청서는 삭제합니다.  
- **순서변경**: 입력 항목의 표시 순서를 조정합니다.  
- **주소찾기**: 우편번호 검색 기능으로 정확한 주소를 입력합니다.  
- **기본정보 입력**: 이름, 연락처, 과목 등 신청에 필요한 정보를 작성합니다.  
- **서약서 입력**: 신청에 필요한 서약 내용을 기재합니다.  
- **수정 가능 여부 설정**: 신청자가 추후 내용을 수정할 수 있는지 여부를 지정합니다.  
- **사용 여부 설정**: 신청서를 실제로 사용할지(사용/미사용) 결정합니다.  
- **등록일 확인**: 신청서 등록 일자를 확인합니다.

- 수강신청서 입력 항목은 관리자가 직접 구성할 수 있습
